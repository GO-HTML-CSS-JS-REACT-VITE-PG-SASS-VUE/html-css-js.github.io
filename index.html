Информация о проекте

Этот проект демонстрирует работу с 3D трансформациями в CSS и управление ими через JavaScript.

Используемые технологии:

  • CSS 3D Transforms: transform-style: preserve-3d, perspective, rotateX, rotateY, rotateZ, translateZ
  • JavaScript DOM Manipulation: Изменение CSS свойств через JS, обработка событий
  • CSS Variables: Динамическое изменение размера куба и перспективы
